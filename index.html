Fixed Background Images

A jQuery plugin to fix background image while scrolling through content of a page. Once new page is scrolled to then a background image changes to the new one of a new page

Download .zip Download .tar View on Github

How to include

Simply include jquery.js and jquery.flip.js you have downloaded

<script src="http://code.jquery.com/jquery-[latest].min.js">
<script src="jquery.flip.js">
